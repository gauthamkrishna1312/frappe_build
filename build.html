Frappe Buildathon Contest Introduction



Project Idea #1: Build an e-commerce platform using Frappe.

  • The platform should allow users to browse and search for products,add them to a cart, and checkout securely.
  • An admin panel for managing products, orders, and sales would be useful.
  • The project must be accompanied by good documentation and a clear README file.
  • The evaluation process will consider both the project and its documentation.

Project Idea #2: Develop a social media platform using Frappe.

  • Users should be able to create profiles, post updates, share media files, and connect with others.
  • Features like likes, comments, and private messaging could be added to make the platform more engaging.
  • The project must be accompanied by good documentation and a clear README file.
  • The evaluation process will consider both the project and its documentation.
